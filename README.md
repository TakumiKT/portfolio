\## サービス概要

登録販売者向けの店舗業務支援アプリ



\## このサービスへの思い・作りたい理由

ドラッグストアで副店長として働く中、多くのOTC医薬品(一般用医薬品)の相談をお客様から受けました。

登録販売者資格者として接客から得られましたが、シフトの関係で他資格者との情報共有や教育を円滑に実施する事が

困難でした。その経験からドラッグストア等の小売店においての、登録販売者・スタッフ向けに

医薬品情報・接客事例・注意点を共有できるアプリを作成したいと考えに至りました。



\## ユーザー層について

本アプリは、OTC医薬品取り扱い店舗の登録販売者および従業員が利用することを想定した業務支援アプリです。

OTC医薬品に関する接客の知見や意見の共有、教育を効率的に実施する事を目的にしています。



\## サービスの利用イメージ

店舗内PCで使用する事を想定しています。

ユーザーはログイン後、接客時の対応内容を「接客メモ」として登録します。

接客メモには、症状の概要・対応時の注意点・補足情報などを自由記述で残せます。

投稿されたメモは、タグや検索機能を用いて後から参照できます。

他のスタッフは過去の事例を確認することで、類似ケースへの対応を振り返ることができます。



\## ユーザーの獲得について

店舗責任者やその上長に、店舗PCへアプリケーションを導入する権限が有ると想定しています。

その上で個人利用ではなく店舗単位での導入し、登録販売者・スタッフが日常業務の中で利用する形を想定し開発する予定です。



\## サービスの差別化ポイント・推しポイント

* Microsoft Teams
* Notion

    \* これら2つのアプリは幅広い用途に対応できる一方で、

特定の情報を振り返ろうとした際に、見つける事に手間取る等の課題があります。

本は、登録販売者の接客判断に必要な情報を

あらかじめ構造化した形式で記録・共有することで、

現場で振り返りやすい知識として蓄積できる点を特徴としています。
